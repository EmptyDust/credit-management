## 🚀 技术栈

### 后端技术
- **语言**: Go 1.21+
- **框架**: Gin Web Framework
- **数据库**: PostgreSQL 15
- **ORM**: GORM
- **认证**: JWT
- **容器**: Docker

### 前端技术
- **框架**: React 18
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **组件库**: Shadcn/ui
- **构建工具**: Vite
- **状态管理**: React Context

### 部署技术
- **容器编排**: Docker Compose
- **云原生**: Kubernetes
- **数据库**: PostgreSQL
- **反向代理**: Nginx
